# BKFC Event Background Assets

This folder is reserved for future approved BKFC event photography used as subtle landing page background imagery.

## Folder Purpose

- `hero/`
  Used for future main hero or top-page background atmosphere images.

- `fighter-pathway/`
  Used for future images behind the Fighter Pathway / fighter development section.

- `global-network/`
  Used for future images behind the global expansion / BKFC network / ecosystem section.

- `event-pulse/`
  Used for optional future event strip thumbnails, upcoming-event visuals, or event-related visual assets.

## Image Guidelines

- Use only approved BKFC, LG Sports, or internal event photos.
- Prefer wide arena shots, ring shots, crowd shots, walkouts, silhouettes, lighting, and production atmosphere.
- Avoid random third-party images.
- Avoid unapproved media photos.
- Avoid overly graphic injury images.
- Avoid sponsor-heavy images unless approved.
- Avoid low-quality screenshots.
- Avoid photos that make overlay text difficult to read.
- Recommended background image width: `1800-2400px`.
- Recommended formats: `.jpg` or `.webp`.
- Recommended file size: ideally under `400-700 KB` per background image.
